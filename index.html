1. Negative Transformation

Inverts intensity values. Useful for highlighting white details in dark regions.

\( s = 255 - r \)

2. Log Transformation

Expands dark pixels and compresses bright pixels. Enhances low-intensity regions.

\( s = c \log(1 + r) \)

3. Gamma Transformation

Adjust brightness using gamma correction. Gamma < 1 brightens, Gamma > 1 darkens.

\( s = 255 \left(\frac{r}{255}\right)^{\gamma} \)

2

4. Thresholding

Converts grayscale image into binary image based on selected threshold value.

\[ s = \begin{cases} 0 & r < T \\ 255 & r \ge T \end{cases} \]

127

5. Contrast Stretching

Expands narrow intensity range to improve image contrast.

\( s = \frac{(r - r_{min})}{(r_{max} - r_{min})} \times 255 \)
