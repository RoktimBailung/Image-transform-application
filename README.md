# Image Intensity Transformation Studio

A full-stack web application that demonstrates classical image processing intensity transformation techniques with real-time visualization.

## 🚀 Live Demo
https://image-transform-application.onrender.com/

## 📌 Features

- Negative Transformation
- Log Transformation
- Gamma Correction (Interactive Slider)
- Thresholding
- Contrast Stretching
- Histogram Visualization (Before & After)
- Download Processed Image
- Reset Image
- Dark Mode Toggle

## 🛠 Tech Stack

- Python (Flask)
- OpenCV
- Matplotlib (Agg Backend)
- JavaScript (Fetch API)
- HTML5 + CSS3
- Gunicorn
- Deployed on Render
